What is Commup?

Jekyll is a simple, wharehouse management, stock allocation, sales and payment system. It takes very low affort to learn how to use and operate the application.

This ready to use application can increase your productivity and improve sales by helping you to manage all the details without having to worried about small mistake that can make big problem. Start using now It's Free

Easy to install and Operate

Just restore database to SQL Server Express, Copy the files in web server then you ready to go.

Use Anywhare

Commup is a webbase application. So you can run it, operate it, monitor it and control it any where. You can use your pc, tablet or even phone.
